#vue-init [![npm package](https://img.shields.io/npm/v/vue-cli.svg)](https://www.npmjs.com/package/vue-init)

> A simple template for Vue2 projects.

### Installation

``` bash
$ npm install vue-init
```

### Usage
``` bash
$ node ./node_modules/vue-init/bin/init.js
```
